# The auctionhousebot table

Contains items that will be sold by the auction house bot.

## Structure

### 1. item - int(11) unsigned

Item id from the `item_template` table.

### 2. stack - tinyint(3) unsigned

Stack size.

### 3. bid - int(11) unsigned

Bid price in copper.

### 4. buyout - int(11) unsigned

Buyout price in copper.
